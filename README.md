# Airline Cost Analytics ETL Pipeline

## Project Overview
Designed and implemented an end-to-end ETL pipeline to process airline shipment and logistics cost data using Azure Data Factory, Azure Databricks, PySpark, SQL, and Power BI.

## Tech Stack
- Azure Data Factory
- Azure Databricks
- PySpark
- Azure Data Lake
- SQL
- Power BI

## Business Problem
Manual airline logistics reporting caused delays and inconsistent cost calculations.

## Solution
Built an automated ETL pipeline to ingest, transform, and visualize airline shipment and cost data.

## Features
- Automated data ingestion
- Data cleansing and validation
- Cost reduction calculations
- KPI reporting
- Power BI dashboard integration

## Architecture
Raw CSV Files → Azure Data Factory → Azure Data Lake → Databricks (PySpark) → SQL Analytics → Power BI Dashboard

## KPIs
- Total Shipment Cost
- Cost Reduction %
- Airline-wise Spend
- Monthly Logistics Trends

## Results
- Reduced manual effort by 80%
- Improved reporting accuracy
- Automated multi-source data processing
